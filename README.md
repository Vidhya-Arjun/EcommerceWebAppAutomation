E-commerce Web Automation  – Selenium + Pytest + Behave BDD and integrated with allure report

This project is an end-to-end UI automation framework built using Python, Selenium WebDriver, and Pytest to automate the Saucedemo application.

The framework follows the Page Object Model (POM) design pattern to ensure maintainability, scalability, and reusability of automation code.

Data-driven login validation
